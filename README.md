## QuickStart
### 1. Запускаем STM32 CubeIDE (Нужно нажать на соотвествующую иконку).
![First](https://imgur.com/l6FsyB5) 
### 2. Создаем проект.
![Second](https://imgur.com/usstVgB)
### 3. Выбираем тип тактового генератора.
![Third](https://imgur.com/O69p2Cd)
### 4. Далее переходим в Clock Configuration и выставляем параметры как на картинке.
![Fourth](https://imgur.com/Fk46397)
### 5. Выбираем PC13 (Нажав на него левой кнопкой мыши).
![Fifth](https://imgur.com/LMyzIgk)
### 6. Выбираем GPIO_Output в открывшемся меню.
![Six](https://imgur.com/Hkc1RNQ)
### 7. Открываем настройки GPIO.
![Seven](https://imgur.com/Ysxe0wq)
### 8. Выставляем настройки как на картинке.
![Eigth](https://imgur.com/cpgwKum)
